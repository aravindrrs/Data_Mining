# Q1)
# Classification: classification is a process of labelling  discreet data. The output is usually finding the class of a given datum.
# This method is mainly for catagorical data types. Using a given set of data with details (training set) we develop models
# which can be used to identify the label or the group of ta discrete test datum. for eg: catagorical analysis of a set of population
# in a given city can be labelled as "rich","super rich","middle income","poor"
        
# Regression is mainly for continuos numerical data. It is used to predict a missing value based on the trends of the available values.
# it is used to predict the output response based on the training set. for eg: based on the increasing prices of a plot of land in a 
# given city from the year 1980-2015, we can predict its approximate price in the year 2019.

#Both classification and regression are similar in the sense they are used for predictions based on the knowledge of an available set 
# samples/ training sets. They both involve creating generic models to predict the output of a test variable.
